# Java Final Exam Preparation Questions

This file contains all the quiz questions that are prepared for Java final exam.

1. **What is Java?**
   - Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

2. **Explain the concept of Object-Oriented Programming in Java.**
   - Object-Oriented Programming (OOP) is a paradigm that organizes software design around data, or objects, rather than functions and logic.

3. **What are the main concepts of OOP in Java?**
   - Encapsulation
   - Inheritance
   - Polymorphism
   - Abstraction

4. **What is the difference between JDK, JRE, and JVM?**
   - JVM (Java Virtual Machine): It is an abstract machine, a specification that provides a runtime environment in which Java bytecode can be executed.
   - JRE (Java Runtime Environment): It is the implementation of JVM and provides libraries and classes.
   - JDK (Java Development Kit): It provides JRE and development tools.

5. **Explain the concept of Garbage Collection in Java.**
   - Garbage Collection is the process by which Java programs perform automatic memory management by reclaiming memory from objects no longer in use.

6. **What are Java Exception Handling keywords?**
   - **try:** The "try" block contains a set of statements that might throw an exception.
   - **catch:** The "catch" block handles the exception.
   - **finally:** It is used to execute the necessary code of the program after a try-catch block regardless of an exception.
   - **throw:** Keywords for explicit throwing of an exception.
   - **throws:** Specifies exceptions that a method can throw.

7. **What is the use of the 'this' keyword in Java?**
   - It is a reference variable that refers to the current object.

8. **Explain the 'final' keyword in Java.**
   - When a variable is declared as final, it cannot be modified after initialization.
   - A final method cannot be overridden.
   - If a class is final, it cannot be inherited.

9. **What is the purpose of Java Inner Classes?**
   - It is used to logically group classes and interfaces in one place to make code more readable and maintainable. 

10. **What are default methods in an interface in Java?**
   - It is a method that has a body in an interface. These methods can have default behaviour predefined.

11. **What is a Singleton Class in Java?**
   - A class is a singleton when there is a single instance of the class that provides a way to access that instance.
